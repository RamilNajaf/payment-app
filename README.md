## Small Banking Application

## Tech Stack

- Gradle
- Spring Framework
- Java 17
- Kafka
- Junit
- Mockito
- Liquibase
- Mysql
- Docker

## Microservices

- gateway
- user-ms
- ms-account
- ms-payment

## For running microservice

- **docker-compose up**
- **./gradlew bootRun** for each microservice( run it inside microservices)

## Features

**_USER-MS_**

- REGISTER WITH USERNAME AND PASSWORD
- LOGIN WITH USERNAME AND PASSWORD

**_MS-ACCOUNT_**

- CREATE ACCOUNT
- SEE YOUR ACCOUNT
- GET YOUR TRANSACTIONS

**_MS-PAYMENT_**

- TOP-UP
- PURCHASE
- REFUND
